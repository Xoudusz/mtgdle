# MTGdle — Claude Instructions

## Project
MTG daily guessing game with 3 modes: Carddle (`/classic`), Artdle (`/art`), Flavordle (`/flavor`).
Stack: Next.js 14 (App Router), TypeScript, Tailwind CSS, Scryfall card data, PocketBase for stats.
